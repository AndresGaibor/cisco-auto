### 3. Output Formats ⚠️ FAIL

```
COMMAND: cisco-auto device list --json labs/vlan-basico.yaml
EXPECTED: JSON válido parseable
ACTUAL: ❌ IGNORED - El flag --json es ignorado, siempre devuelve texto

COMMAND: cisco-auto device list --output yaml labs/vlan-basico.yaml
EXPECTED: YAML válido
ACTUAL: ❌ IGNORED - El flag --output yaml es ignorado

COMMAND: cisco-auto device list --output table labs/vlan-basico.yaml
EXPECTED: Tabla con columnas alineadas
ACTUAL: ❌ IGNORED - El flag --output table es ignorado

COMMAND: cisco-auto -j device list labs/vlan-basico.yaml
EXPECTED: JSON válido
ACTUAL: ❌ IGNORED - El flag global -j es ignorado
```

**ROOT CAUSE**: El comando `device list` (y probablemente otros) no utiliza los global flags definidos en `flags.ts`. Hace `console.log()` directo sin pasar por `formatOutput()`.
