## Funcionalidades Implementadas

### Tipos
- `ToolResult<T>` - Resultado exitoso de una tool
- `ToolError` - Resultado fallido de una tool
- `ToolResultMetadata` - Metadatos (duration, itemsCount, etc.)
- `ToolOutputFormat` - Formatos: json, table, text
- `ToolFormatterOptions` - Opciones de formateo

### Formateadores
1. **JSON**: Salida completa con metadatos
2. **Table**: Resumen tabular con encabezados y filas
3. **Text**: Mensaje legible con iconografía

### Funciones Exportadas
- `formatToolResult()` - Formatea ToolResult
- `formatToolError()` - Formatea ToolError
- `formatToolOutput()` - Auto-detecta éxito/error
- `createToolResult()` - Factory para resultados
- `createToolError()` - Factory para errores

### Features
- Extracción automática de itemsCount
- Formateo de duración legible (ms, s, m)
- Soporte para warnings
- Mensajes user-friendly
- Sugerencias de corrección
- Causa raíz (verbose mode)
- Colores usando chalk (configurables)
